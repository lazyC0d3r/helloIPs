<h1 align="center"> helloIPs - A Simple Network IP checking Tool </h1>
>`helloIPs` is a simple tool to display the Public and Private IP of Your System
## Installation

```bash
$ git clone https://github.com/lazyC0d3r/helloIPs.git
```
```bash
$ cd helloIPs
```
```bash
$ chmod +x helloIPs
```
```bash
./helloIPs
```
